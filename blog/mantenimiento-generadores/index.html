Mantenimiento preventivo de generadores: Guía completa

KOR Generadores · 2026

Cambio de aceite

El aceite debe cambiarse cada 100-200 horas de uso o al menos una vez al año. Usá siempre el aceite recomendado por el fabricante (generalmente SAE 10W-30 o 15W-40).

Filtros

Revisá el filtro de aire cada 50 horas. Reemplazá el filtro de aceite con cada cambio de aceite. Los generadores diesel también necesitan cambio de filtro de combustible.

Batería

Verificá el nivel de carga mensualmente. Mantené los bornes limpios. Reemplazá cada 2-3 años.

Ejercicio regular

Si el generador no se usa frecuentemente, hacelo funcionar 15-30 minutos cada mes para mantener los componentes lubricados.

Cuándo llamar a un técnico

Si notás humo excesivo, ruidos anormales, caídas de potencia o dificultad para arrancar, contactá a servicio técnico.
